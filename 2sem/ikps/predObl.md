# Описание предметной области

**Предметная область: система управления заказами интернет-магазина.** 

Модель данных включает ключевые сущности: Клиенты, Товары и Заказы. Для описания комплексных данных используются типы: Адрес, Элемент заказа, Способ оплаты и Телефон. Связи между сущностями отражают типичные бизнес-процессы: клиенты делают заказы, которые содержат несколько товаров.

# Описание файлов

DTD: внешний файл, параметрические и общие сущности, атрибуты (ID, IDREF, CDATA, перечисления), внешняя неанализируемая сущность.

XML Schema: пространство имен, сложные типы, наследование, модельные группы (sequence, choice, all), фацеты, ключи и ключевые ссылки.

XML-пример: соответствует обеим схемам, содержит разнообразные данные, ссылается на DTD и XSD.
